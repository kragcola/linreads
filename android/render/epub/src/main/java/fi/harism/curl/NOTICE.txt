The page-curl OpenGL implementation (CurlView / CurlRenderer / CurlMesh / CurlPage) is
used as the engine behind PageFlipStyle.SIMULATION (仿真翻页). Only the demo Activity was
omitted. Files are kept verbatim with their original Apache headers, with ONE modification:
CurlView adds a `CurlAnimationObserver` hook (a callback fired when a curl animation settles)
so the EPUB engine can commit/clean up the page turn. The change is marked inline with
"LinReads addition" comments. The Apache License 2.0 permits modification under §4.

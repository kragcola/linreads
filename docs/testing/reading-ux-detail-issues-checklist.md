**复现场景**:
```

操作: 翻页模式阅读
现象: 图片显示尺寸明显小于预期
```

**需要检查的变体**:
- [ ] 内联图片 (`<img>` 标签)
- [ ] 块级图片 (`<figure>` 包裹)
- [ ] Base64 内联图片
- [ ] 外链图片 (file:// 路径)
- [ ] SVG 图片
- [ ] 不同宽高比图片 (正方形/横图/竖图)

**对比测试**:
- 静读天下横版图片显示如何？
- Librera 横版图片显示如何？
- 竖版vs横版图片大小差异

**根因分析方向**:
```kotlin
// android/render/epub/src/main/kotlin/dev/readflow/render/epub/EpubReflowEngine.kt

// 2. 横版时 contentWidth 计算错误
// 3. 图片缩放比例未考虑orientation
// 4. RecyclerView item宽度限制
```

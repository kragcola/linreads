# Android v4 Pure Reading Gap Checklist

_Last updated: 2026-06-27_

Purpose: one checklist for closing and backfilling Android v4 pure-reading gaps.
Update this file whenever a gap is implemented, verified, deferred, or removed from
scope.

## Scope

Included:

- Local reading flow: import/open/read/continue.
- Core reader UX: navigation, gestures, layout controls, TOC, search, bookmarks,
  highlights, progress, accessibility, performance.
- Reading data control: progress/bookmark/annotation persistence, backup/export,
  offline cache, optional Calibre source.
- Optional pure-reading formats: DOCX/CBZ.

Excluded from this checklist:

- TTS.
- Ink/handwriting implementation.
- Reading stats.
- OPDS extension work unless it becomes part of the core reading-source flow.

## Status Legend

- `TODO`: not implemented.
- `PARTIAL`: code exists but does not meet the v4 acceptance contract.
- `VERIFY`: code path exists, but device/performance/accessibility smoke is still missing.
- `DONE`: implemented and evidence is recorded.
- `DEFERRED`: explicitly postponed or removed from current milestone.
